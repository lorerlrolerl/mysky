# App Icon Setup

## Quick Setup

1. Open `assets/icon.svg` in a design tool or browser
2. Export as PNG in these sizes:
   - 48x48 (mdpi)
   - 72x72 (hdpi)
   - 96x96 (xhdpi)
   - 144x144 (xxhdpi)
   - 192x192 (xxxhdpi)
3. Replace icons in `android/app/src/main/res/mipmap-*/ic_launcher.png`
